Chess-Playing Robotic Arm

A vision-guided robotic arm that plays chess against you — and teaches you while doing it. From inverse kinematics to power management, the whole system was engineered from the ground up.

Role
System Engineer (team project)
Timeline
Sep – Dec 2024
Stack
Python · C/C++ · ROS · Gazebo · Rviz · Raspberry Pi
Recognition
IEEE SMC 2025 Publication · 1st Prize IEEE TSYP

The idea

Chess is a great teacher — but a screen can only teach so much. We built a physical robotic tutor: an arm that perceives the board through a camera, decides its move, and physically picks up and places the pieces, explaining the game as it goes. The goal was a system reliable enough to run full games unattended.

What I engineered

  • Vision-guided robotic system with real-time simulation and trajectory planning, built as a custom ROS package using Gazebo and Rviz — every move is validated virtually before the physical arm executes it.
  • Inverse kinematics algorithms that convert target Cartesian coordinates on the board into precise joint values, achieving the repeatability needed to grip and place small chess pieces square after square.
  • Dual-function power management system allowing the robot to run from mains power while simultaneously charging its batteries — no downtime between games.

Results

  • Won 1st prize at the IEEE TSYP Congress chess robot challenge (Dec 2024).
  • The work was peer-reviewed and published as “Intelligent Chess Robot Tutor” at IEEE SMC 2025.
